A simple and lightweight web framework for Bun

## Automatic Installation

### Initialize Project

```bash
bun create kurogashi [project-name]
```

## Breaking Down

This will create a new project with the following structure:

<Files>
    <Folder name="src" defaultOpen>
        <Folder name="routes" defaultOpen>
            <File name="index.ts" />
        </Folder>
    </Folder>
    <File name="package.json" />
    <File name="tsconfig.json" />
    <File name="kurogashi.config.ts" />
    <File name=".gitignore" />
</Files>

The `package.json` will contain two scripts `build` and `dev`:

### Building your app

Read more about build and presets [here](/docs/build)

### Running your app

Running this script will spin up a development server with hot reload.

Dev command uses default `bun` preset

## Chaning port
You can either change the port in the `kurogashi.config.ts` (server.port) or by setting the `PORT` environment variable.
